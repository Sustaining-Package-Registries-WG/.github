# Sustaining Package Registries Working Group

Open source package registries are critical infrastructure for the global software ecosystem. The **Sustaining Package Registries Working Group**, hosted by the Linux Foundation, brings together leaders from major public package ecosystems and other stakeholders to collaborate on economic sustainability, governance enablement, and collective messaging and transparency of sustaining community software repositories at global scale.

## Members

- Alpha-Omega
- Linux Foundation
- OpenJS Foundation
- Open Source Security Foundation (OpenSSF)
- PHP Foundation (Packagist / Composer)
- Python Software Foundation (PyPI)
- Ruby Central (RubyGems)
- Rust Foundation (crates.io)
- Sonatype (Maven Central)

## Governance

The working group operates by consensus, with participation limited to technical and operational leaders of public-facing community software repositories.
